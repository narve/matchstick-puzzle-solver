The essential matchstick equation toolbox :)

Made by Narve. Comments and feedback welcome on the Github project page.

Enter your puzzle in the input box, or click on any equation to analyze it. If the equation is incorrect, any solutions will be displayed. If the equation is correct, possible puzzles (simple mutations) will be displayed.

Only tested in newest Chrome and Firefox. Chrome has some font issues so the input box isn't as nice as in Firefox :)

Note that 7 really only uses two matches, and that * means multiply. If you know of a more suitable font to use, please let me know!
